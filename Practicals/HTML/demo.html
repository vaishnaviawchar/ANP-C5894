JAVA INTRODUCTION

Java technology is a language of programming as well as a platform.
Java is a high level, robust, secured and object-oriented programming language.
And any hardware or software environment in which a program runs is known as a platform.
Since Java has its own runtime environment (JRE) and API, it is called a platform.
