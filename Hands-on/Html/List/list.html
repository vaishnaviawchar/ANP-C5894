Java OOPs Concepts

  1. Object

    2. Any entity that has state and behavior is known as an object

  2. Class

    3. Collection of objects is called class. It is a logical entity.

  3. Inheritance

    4. When one object acquires all the properties and behaviors of a parent object, it is known as inheritance.

  4. Polymorphism

    5. If one task is performed in different ways, it is known as polymorphism.

  5. Abstraction

    6. Hiding internal details and showing functionality is known as abstraction.

  6. Encapsulation

    7. Binding (or wrapping) code and data together into a single unit are known as encapsulation.
